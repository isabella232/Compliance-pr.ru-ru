<span data-ttu-id="1ab6e-p102">Согласно общему регламенту по защите данных (GDPR) управляющим данными требуется подготовить оценку влияния на защиту данных (DPIA) для операций обработки, в результате которых "может появиться высокий риск нарушения прав и свобод физических лиц". У службы обработчика данных для Windows Enterprise нет характеристик, требующих создания DPIA с помощью управляющего данными. Rather, whether a DPIA is required will be dependent on the details and context of how the data controller deploys, configures, and uses the data processor service for Windows Enterprise.</span></span>
