# <a name="data-protection-impact-assessments-guidance-for-data-controllers-using-microsoft-data-processor-service-for-windows-enterprise"></a>Оценка влияния на защиту данных: руководство для управляющих данными, использующих службу обработчика данных Microsoft для Windows Enterprise

>[!NOTE]
>Эта статья предназначена для участников программы предварительной оценки службы обработчика данных для Windows Enterprise и требует принятия определенных условий. Чтобы узнать больше о программе и согласиться с условиями использования, см. [https://aka.ms/WindowsEnterprisePublicPreview](https://aka.ms/WindowsEnterprisePublicPreview).

Согласно общему регламенту по защите данных (GDPR) управляющим данными требуется подготовить оценку влияния на защиту данных (DPIA) для операций обработки, в результате которых «может появиться высокий риск нарушения прав и свобод физических лиц». У службы обработчика данных для Windows Enterprise нет характеристик, требующих создания DPIA с помощью управляющего данными. Необходимость DPIA зависит скорее от того, как управляющий данными развертывает, настраивает и использует службу обработчика данных для Windows Enterprise. 

Цель этого документа — предоставить управляющим данными информацию о службе обработчика данных для Windows Enterprise, которая поможет им определить, требуется ли оценка DPIA, и если да, то какие сведения необходимо в нее включить.

>[!Note]
>Корпорация Майкрософт не предоставляет никаких юридических советов в этом документе. Этот документ предоставляется исключительно в информационных целях. Клиентам рекомендуется сотрудничать со своими уполномоченными сотрудниками по вопросам безопасности и юрисконсультами, чтобы определить необходимость и содержимое DPIA, связанных с использованием службы обработчика данных Windows Enterprise или другой веб-службы Майкрософт.

## <a name="part-1-determining-whether-a-dpia-is-needed"></a>Часть 1. Определение необходимости DPIA

Статья 35 регламента GDPR требует, чтобы управляющий данными составил оценку влияния на защиту данных (DPIA), «где тип обработки c использованием новых технологий и учетом характера, масштаба, контекста и целей обработки может представлять высокий риск для прав и свобод физических лиц». Кроме того, в ней указаны определенные факторы, представляющие высокий риск. Эти факторы описаны в таблице ниже. При определении необходимости в оценке DPIA управляющий данными должен учитывать эти, а также другие значимые факторы, в свете своих конкретных реализаций и применения службы обработчика данных для Windows Enterprise.

## <a name="table-1-data-processor-service-for-windows-enterprise-dpia-risk-factors"></a>Таблица 1. Факторы риска DPIA службы обработчика данных для Windows Enterprise 

|**Фактор высокого риска**|**Важная информация о службе обработчика данных для Windows Enterprise**|
|:----|:----|
| Системная и подробная оценка личных характеристик физического лица, которая основана на автоматизированной обработке, в том числе составлении психологического портрета, и на которой основаны решения, порождающие юридические последствия в отношении этого лица или влияющие на него похожим образом. |Служба обработчика данных для Windows Enterprise не предоставляет возможность выполнять определенную автоматизированную обработку данных.<br><br> Однако, так как в качестве источника данных в других службах используется служба обработчика данных для Windows Enterprise, управляющий данными может настроить такие службы для обработки. Управляющие принимают такое решение, исходя из своего способа использования служб, подключенных к службе обработчика данных для Windows Enterprise.|   
| Масштабная обработка* особых категорий данных (персональные данные, раскрывающие расовое или этническое происхождение, политические взгляды, религиозные или философские убеждения или членство в профсоюзах и обработка генетических и биометрических данных с целью однозначной идентификации физического лица, данных о здоровье или данных, касающихся его сексуальной жизни или ориентации) или персональных данных, касающихся судимости или правонарушений. | Служба обработчика данных для Windows Enterprise не разрабатывалась специально для обработки особых категорий персональных данных, и ее использование не увеличивает риск, присущий обработке со стороны управляющего.<br><br> Однако управляющий данными может использовать службы, подключенные к службе обработчика данных для Windows Enterprise, чтобы обработать перечисленные особые категории данных. Службы, которые используют службу обработчика данных для Windows Enterprise в качестве источника данных, позволяют клиенту отслеживать или иным образом обрабатывать любые данные, в том числе особые категории персональных данных. Однако корпорация Майкрософт как обработчик данных не имеет контроля над таким использованием и имеет слабое представление о нем либо не имеет его вообще. Управляющий данными обязан определять надлежащее использование своих данных. |

## <a name="part-2-contents-of-a-dpia"></a>Часть 2. Содержимое DPIA 

Статья 35(7) требует, чтобы оценка влияния на защиту данных определяла цели и системное описание предполагаемой обработки. Системное описание всесторонней оценки DPIA может включать такие факторы, как типы обрабатываемых данных, продолжительность хранения данных, места размещения и передачи данных и третьи стороны, которым может быть предоставлен доступ к этим данным. Кроме того, DPIA должна включать: 

оценку необходимости и пропорциональности операций обработки по отношению к целям; 

оценку рисков для прав и свобод физических лиц; и 

меры разрешения этих рисков, включая гарантии безопасности, меры предосторожности и механизмы для обеспечения защиты персональных данных и демонстрации соблюдения настоящих Правил с учетом прав и законных интересов субъектов данных и других заинтересованных лиц. 

В таблице ниже приведены важные сведения о службе обработчика данных для Windows Enterprise, которые относятся к каждому из элементов. Как и в первой части, управляющие данными должны рассматривать факторы, представленные в таблице, а также другие связанные факторы в контексте конкретных реализаций и способов использования службы обработчика данных для Windows Enterprise.
